# Release Checklist

Before sharing a build:

- Python files compile.
- Unit tests pass.
- The web app builds locally.
- The smoke test runs.
- One direct extraction succeeds.
- Client benchmark completes.
- Manual validation file is generated when manual scoring is required.
- No runtime tracebacks are present in benchmark outputs.
- Database save is tested when SQL Server is available.
- Documentation does not contain local-only paths or internal patch labels.
